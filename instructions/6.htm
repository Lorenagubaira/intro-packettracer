Connecting end-devices

To finish our network, we need to connect the rest of the devices that we have on the screen. To do this, we will connect the cables with the Automatic Connection and configure the devices to receive a dynamic IP from the server that we configured in the previous step.

On "PC 0"

  1. Clicking on the PC will show the configuration options in the "Config" tab.
  2. In the left panel, you will see the "FastEthernet0" interface.
  3. You will have to activate the interface with the "Status port" checkbox.
  4. In the "IP Configuration" section, select the "DHCP" option to receive a dynamic IP from the server.

Conectando dispositivos finales

Para terminar nuestra red, debemos conectar el resto de los dispositivos que tenemos en pantalla. Para ello conectaremos los cables con la Conexión automática y configuraremos los equipos para recibir una IP dinámica desde el servidor que configuramos en el paso anterior.

En la "PC 0"

  1. Haciendo click en la PC veremos las opciones de configuración en la pestaña "Config"
  2. En el panel izquierdo podremos ver la interfaz "FastEthernet0"
  3. Tendrás que activar la interfaz con la casilla de verificación "Status port"
  4. En la sección "IP Configuration" selecciona la opcion "DHCP" para recibir una IP dinámica del servidor

Connecting the PC to the switch

  1. To use the auto-connect feature, we will select the orange lightning bolt icon in the "Connections" section.

Conectando la PC al switch

  1. Para esto utilizaremos la función de conexión automática, para esto seleccionaremos el icono del rayo naranja en la sección "Connections"
  1. When the cursor changes, we can click on the server and then on the PC to create the link. This tool automatically determines the correct link type and selects a compatible interface.
  1. Cuando el cursor cambie, podremos hacer click en el servidor y luego en el PC para hacer el enlace. Esta herramienta determina automáticamente el tipo de enlace correcto y selecciona una interfaz compatible

You must repeat this process of configuration and connection to the switch with all the other end devices on the network.

Debes repetir este proceso de configuración y conexión al switch con todos los demás dispositivos finales de la red.
